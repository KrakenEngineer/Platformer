Жанр: изначально платформер, фактически ближе к головоломке. Двумерный

Строится на (нереалистичном) взаимодействии магнитов. Цель игры - пройти все уровни

Игрок - магнитный шарик

Управление: WASD, пробел = W, Q - магнитить

Содержит:
1) Кнопки - датчики. Датчики выдают сигнал, позволяющий пройти уровень
2) Порталы, ускорители, электромагнитный поток - средства передвижения
3) Магнитные, немагнитные кубы, поверхности
4) Главное меню с уровнями, поддерживается сохранение прогресса
5) Туториал, небольшой первый уровень, второй пуст

Планы:
1) Больше уровней, датчиков, механик в целом
2) Озвучка и улучшение визуальной составляющей, дополнение главного меню
3) Багфиксы по необходимости (сейчас багов не наблюдается)

Все скрипты лежат в Assets/Scripts
